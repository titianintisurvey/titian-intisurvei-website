# PT. TITIAN INTI SURVEI - Next.js Website

## Cara menjalankan

1. Buka folder project ini di Visual Studio Code
2. Jalankan terminal
3. Ketik:

```bash
npm.cmd install
npm.cmd run dev
```

4. Buka browser ke:

```bash
http://localhost:3000
```

## Halaman yang tersedia

- Beranda: `/`
- Layanan: `/services`
